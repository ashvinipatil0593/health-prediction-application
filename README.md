# Health Prediction Application

## Overview

Health Prediction Application is a Streamlit-based web application that allows users to manage patient records and generate AI-powered health remarks using Gemini AI.

## Features

* Add Patient
* View Patient Records
* Update Patient Records
* Delete Patient Records
* Input Validation
* AI Health Prediction using Gemini AI
* SQLite Database Storage

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas
* Gemini AI

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python -m streamlit run app.py

## Project Workflow

Patient Data → Validation → Gemini AI → Health Remark → SQLite Database
